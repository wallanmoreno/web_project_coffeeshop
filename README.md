# Coffee Shop

## 📌 Descrição do Projeto

Este projeto é uma página web de uma cafeteria (Coffee Shop), desenvolvida como parte dos estudos de HTML e CSS avançado.

A página apresenta:

- Um cabeçalho com informações da cafeteria
- Seção de receitas com vídeos
- Formulário para reserva de mesas
- Rodapé com informações e redes sociais

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Metodologia BEM (Block Element Modifier)
- Flexbox
- Normalize.css

---

## 🚀 Funcionalidades

- Navegação entre seções através do menu
- Layout responsivo para telas maiores (>= 1100px)
- Formulário com validação (required)
- Efeitos hover nos links e botões
- Organização de CSS por blocos (arquitetura BEM)

---

## 🔧 Melhorias Futuras

- Tornar o layout responsivo para dispositivos móveis
- Adicionar animações (hover mais avançado, transições)
- Implementar envio real do formulário com JavaScript
